# DingCat OS

陈雪婷的个人空间网站原型。它以蓝黄白手绘风和个人操作系统概念展示产品经验、AI 工作流、真实项目、生活兴趣与联系方式。

## Preview

直接打开 `index.html` 即可查看当前静态原型。

## Structure

- `index.html` - 页面结构
- `styles.css` - DingCat OS 视觉样式
- `app.js` - 粒子标题、导航高亮、拖拽、猫咪提示、项目弹窗和 MBTI 彩蛋交互
- `assets/` - 人物和阿比西尼亚猫贴纸素材
- `design-qa.md` - 设计迭代与检查记录

## Notes

当前版本是中保真个人网站原型，后续可继续补充真实经历内容、压缩图片、接入部署与 SEO。
